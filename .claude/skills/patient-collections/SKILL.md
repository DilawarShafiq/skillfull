# Patient Collections Skill

Expert guidance for collecting patient financial responsibility while maintaining positive relationships.

## When to Use

- Point-of-service collections
- Patient statement management
- Payment plan setup
- Financial assistance screening
- Collection communications

## Patient Responsibility Components

| Type | Description |
|------|-------------|
| Copayment | Fixed amount per visit |
| Coinsurance | Percentage of allowed amount |
| Deductible | Annual amount before insurance pays |
| Non-covered | Services not under plan |
| Self-pay | Uninsured patient responsibility |

## Statement Cycle

| Cycle | Age | Action |
|-------|-----|--------|
| 1st Statement | 0-30 days | Initial notification |
| 2nd Statement | 31-60 days | Reminder + payment options |
| 3rd Statement | 61-90 days | Urgent notice |
| Final Notice | 91-120 days | Pre-collection warning |

## Collection Best Practices

1. Verify insurance and estimate costs upfront
2. Collect copays at check-in
3. Offer multiple payment options
4. Set up payment plans proactively
5. Screen for financial assistance
6. Document all communications

## Compliance Requirements

- **FDCPA**: Fair Debt Collection Practices Act
- **HIPAA**: Verify identity before discussing account
- **No Surprises Act**: Good faith estimates for self-pay
- **State Laws**: Vary by jurisdiction

## KPIs

| Metric | Target |
|--------|--------|
| POS collection rate | >90% |
| Patient A/R days | <45 days |
| Bad debt rate | <3% |
| Payment plan compliance | >80% |
