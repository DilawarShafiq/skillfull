# RCM Analytics Skill

Expert guidance for revenue cycle metrics, reporting, and performance analysis.

## When to Use

- Analyzing RCM KPIs
- Building dashboards
- Benchmarking performance
- Denial trending
- Payer analysis

## Tier 1: Executive KPIs

| Metric | Formula | Target |
|--------|---------|--------|
| Net Collection Rate | Payments / Adjusted Charges | >95% |
| Days in A/R | Total A/R / (Annual Charges / 365) | <35 days |
| Denial Rate | Denied Claims / Total Claims | <5% |
| Bad Debt Rate | Bad Debt / Net Revenue | <3% |

## Tier 2: Operational KPIs

| Metric | Target |
|--------|--------|
| Clean Claim Rate | >95% |
| First Pass Resolution | >90% |
| POS Collection Rate | >90% |
| Appeal Success Rate | >50% |

## Tier 3: Process KPIs

| Metric | Target |
|--------|--------|
| Charge Lag | <3 days |
| Payment Posting Lag | <2 days |
| Coding Accuracy | >95% |
| Authorization Compliance | >98% |

## Standard Reports

### Daily
- Charges and payments
- Deposit reconciliation
- Claim submission volume

### Weekly
- A/R aging summary
- Denial trends
- Productivity metrics

### Monthly
- Complete A/R aging
- Payer mix analysis
- Net collection trending
- Denial root cause analysis

## Dashboard Elements

**Executive View:**
- Net collection rate (gauge)
- Days in A/R (trend)
- Cash collections (bar chart)
- A/R aging (pie chart)

**Operational View:**
- Daily claim volume
- Denial queue
- Productivity metrics
