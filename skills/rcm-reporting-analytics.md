# RCM Reporting & Analytics Skill

## Overview
This skill covers the key metrics, reports, and analytics used to monitor and optimize Revenue Cycle Management performance for US healthcare providers.

## Key Competencies

### Core RCM Metrics

#### Financial Performance Metrics
| Metric | Formula | Target |
|--------|---------|--------|
| Net Collection Rate | (Payments / Adjusted Charges) x 100 | >95% |
| Gross Collection Rate | (Payments / Gross Charges) x 100 | Varies |
| Days in A/R | Total A/R / (Annual Charges / 365) | <35 days |
| A/R > 90 days | (A/R > 90 / Total A/R) x 100 | <15-20% |
| Bad Debt Rate | (Bad Debt / Net Revenue) x 100 | <3% |
| Cost to Collect | (RCM Costs / Net Collections) x 100 | <3-5% |

#### Operational Metrics
| Metric | Formula | Target |
|--------|---------|--------|
| Clean Claim Rate | (Clean Claims / Total Claims) x 100 | >95% |
| First Pass Resolution Rate | (Paid on First Submission / Total Claims) x 100 | >90% |
| Denial Rate | (Denied Claims / Total Claims) x 100 | <5% |
| Appeal Success Rate | (Overturned Appeals / Total Appeals) x 100 | >50% |
| Charge Lag | Days from DOS to Charge Entry | <3 days |
| Coding Accuracy | (Correct Codes / Total Codes) x 100 | >95% |

#### Point-of-Service Metrics
| Metric | Target |
|--------|--------|
| Pre-registration Rate | >80% |
| Eligibility Verification Rate | >98% |
| POS Collection Rate | >90% |
| Authorization Compliance | >98% |

### Standard RCM Reports

#### Daily Reports
- Daily charges and payments
- Bank deposit reconciliation
- Claim submission volume
- Payment posting summary
- Cash collections summary

#### Weekly Reports
- A/R aging summary
- Denial trends
- Collection activity
- Productivity reports
- Outstanding authorizations

#### Monthly Reports
- Complete A/R aging report
- Payer mix analysis
- Denial analysis by category
- Net collection rate trending
- Days in A/R trending
- Bad debt and write-off analysis
- Charge lag report
- Coding accuracy audit

#### Quarterly/Annual Reports
- Revenue cycle scorecard
- Payer contract performance
- Year-over-year comparisons
- Benchmark comparisons
- Strategic initiative progress

### A/R Aging Report Analysis

#### Aging Report Components
- Patient name and account number
- Date of service
- Payer information
- Original charge amount
- Payments and adjustments
- Current balance
- Aging bucket (0-30, 31-60, 61-90, 91-120, 120+)
- Last activity date
- Claim status

#### Analysis Approach
1. Review total A/R and distribution
2. Identify concentration by payer
3. Analyze trends over time
4. Identify specific problem accounts
5. Prioritize follow-up actions

### Denial Analytics

#### Denial Trending
- By reason code
- By payer
- By provider
- By service type
- By department
- Over time

#### Root Cause Analysis
- Identify top denial reasons
- Quantify financial impact
- Determine preventable vs non-preventable
- Trace to source process
- Implement corrective actions

### Payer Performance Analysis

#### Contract Performance
- Compare actual vs expected reimbursement
- Fee schedule compliance
- Payment timeliness
- Denial rates by payer
- Appeal success by payer

#### Payer Mix Analysis
- Percentage of revenue by payer
- Profitability by payer
- Collection rate by payer
- A/R days by payer

#### External Benchmarks
- MGMA data
